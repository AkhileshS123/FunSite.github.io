We are learners. We are coders. We were motivated by our need to keep learning and growing. What's better than combining all those ideas and creating a typing game embedded in a website? That's precisely what we did.

Tools We Used

HTML

Technically, HTML stands for Hypertext Markup Language and it is a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages. Basically, it is used to structure the raw text and content of the website.



CSS

CSS3 stands for Cascading Style Sheet. It is used to add color and style the website. It can control the margins, padding, and various other positioning attributes of the html elements.

Java

Java is a Object Oriented Programming Language. This programming language is already well-established because it can be used on any platform and is very efficient. This language can be used to create games.
